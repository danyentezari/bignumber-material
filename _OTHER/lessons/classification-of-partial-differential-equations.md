### Definitions

**0 $u_{xx}$** is an unknown function

**1 linear differential operator, L**. A differential operator operating on a linear combination of two differentiable functions. This is the same as the linear combination [6, pp. 122].

**2 differential operator** is a function [5, pp. 132] that transforms [6, pp. 121] a differentiable function into another function.

$$F(u)(x)=g(x,u(x),u^{\prime}(x),...,u^{(n)}(x))$$

where 
- $g$ is a continuous function
- $u$ is a function

If $u$ is in $C$, the domain of the $F$. Differential operators are labeled with uppercase letters [6, pp. 121].


**1 linearity** An example is this [6, pp. 5]

$$F(x,y,y^{\prime},...,y^{(n)})=0$$

where 
- F is a real-valued function

### Canonical Forms

To put a mathematical expression in canonical form is to express it in a way that is "conventional"; that is, to write it in the way that "it's usually done."

$$a(x,y)u_{x}+b(x,y)u_{y}+c(x,y)u+d(x,y)=0$$

$$u_{\epsilon}+\gamma(\xi,\eta)u+\delta(\xi,\eta)=0$$
