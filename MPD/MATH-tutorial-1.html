Logarithms

Used for multiplying large numbers more quickly. Developed by John Napier. For example,

8×128=23×27=210=10248 \times 128 = 2^3 \times 2^7 = 2 ^ {10} = 10248×128=23×27=210=1024

You represent the integers you wish to multiply as powers of two (in this example), add up the exponents, and then lookup the table of logarithms to find answer (in this case, 2102^{10}210)

Evaluating base 2 using logarithms of another base (e.g, base eee or base 101010). This technique is called Change of Base (see proof).

log⁡2(x)=ln⁡(x)ln⁡(2)=log⁡e(x)log⁡e(2)\log_2(x) = \dfrac{\ln(x)}{\ln(2)} = \dfrac{\log_e(x)}{\log_e(2)}log2​(x)=ln(2)ln(x)​=loge​(2)loge​(x)​

or

log⁡2(x)=log⁡10(x)log⁡10(2)\log_2(x) =\dfrac{\log_{10}(x)}{\log_{10}(2)}log2​(x)=log10​(2)log10​(x)​

Note that

log⁡e(n)=ln⁡(n)\log_e(n) = \ln(n)loge​(n)=ln(n)



Proof for Change of Base

by=xb^y = x\quad\quad\quad\quad\quad\quad\quad\quadby=x (given)

x=byx = b^y\quad\quad\quad\quad\quad\quad\quad\enspace\enspacex=by (equivalently)

log⁡b(x)=y\log_b(x) = y \quad\quad\quad\quad\quad\quadlogb​(x)=y (definition of logarithm)

Using a different base

log⁡a(by)=log⁡a(x)\log_a(b^y) = \log_a(x)loga​(by)=loga​(x)

y×log⁡a(b)=log⁡a(x)y \times \log_a(b) = \log_a(x)\quad\quady×loga​(b)=loga​(x) (by Power Rule)

y=log⁡a(x)log⁡a(b)y = \dfrac{\log_a(x)}{\log_a(b)}\quad\quad\quad\quad\quad\quady=loga​(b)loga​(x)​ (by divison)

∴log⁡b(x)=log⁡a(x)log⁡a(b)\therefore \log_b(x) = \dfrac{\log_a(x)}{\log_a(b)}\quad\quad\enspace∴logb​(x)=loga​(b)loga​(x)​ (by substitution)
