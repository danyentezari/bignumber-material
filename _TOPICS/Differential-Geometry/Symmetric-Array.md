# Symmetric Array

<i>

**definition [d]** (*Symmetric Array = Symmetric Matrix*) A square array of components $A_{\mu\nu}$ equal to its transpose:

- $A_{\mu\nu} = A_{\nu\mu}$ .

where

- $\mu,\nu$ run over $1,\ldots,n$ (or $0,\ldots,n-1$ in spacetime).
- symmetry cuts the number of independent entries from $n^{2}$ to $n(n+1)/2$ (e.g. $10$ in $4$ dimensions).

</i>

<i>

**definition [d]** (*Symmetric Array = Metrical Matrix*) The array of metric components in a basis $\{e_{\mu}\}$,

- $g_{\mu\nu} = g(e_{\mu}, e_{\nu})$ ,

which is symmetric: $g_{\mu\nu} = g_{\nu\mu}$.

where

- the left index labels the row and the right index the column.
- these components encode the local geometry (and in GR, the gravitational field).
- also called the metrical matrix or the matrix of the first fundamental form.

</i>
