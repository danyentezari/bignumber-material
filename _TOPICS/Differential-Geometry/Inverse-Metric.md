# Inverse Metric

<i>

**definition [d]** (*Inverse Metric = Inverse Metric Tensor = Contravariant Metric Tensor*) The smooth type-$(2,0)$ tensor $g^{\mu\nu}$ that is the matrix inverse of the metric $g_{\mu\nu}$:

- $g^{\mu\nu}\, g_{\nu\rho} = \delta^{\mu}_{\ \rho}$ .

where

- $g_{\mu\nu}$ is a nondegenerate metric (so the inverse exists and is unique).
- $\delta^{\mu}_{\ \rho}$ is the Kronecker delta (identity map).
- $g^{\mu\nu} = g^{\nu\mu}$ (symmetry is inherited).

</i>

<i>

**definition [d]** (*Inverse Metric = Inverse Metric Tensor = Contravariant Metric Tensor*) The type-$(2,0)$ tensor used to raise indices,

- $V^{\mu} = g^{\mu\nu}\, V_{\nu}$ ,

with $g^{\mu\nu}$ defined by $g^{\mu\nu}\, g_{\nu\rho} = \delta^{\mu}_{\ \rho}$.

where

- lowering uses the metric: $V_{\mu} = g_{\mu\nu}\, V^{\nu}$.
- together, $g_{\mu\nu}$ and $g^{\mu\nu}$ implement the musical isomorphism between $T_{p}M$ and $T^{*}_{p}M$.

</i>
