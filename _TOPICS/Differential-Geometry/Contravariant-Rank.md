# Contravariant Rank

<i>

**definition [d]** (*Contravariant Rank = Contravariant Degree*) For a type-$(q,r)$ tensor, the integer $q$: the number of covector (one-form) arguments the tensor accepts,

- $T(\underbrace{\omega^{1},\ldots,\omega^{q}}_{q},\, v_{1},\ldots,v_{r})$ .

where

- $q$ is the first slot in the type label $(q,r)$.
- also called the contravariant degree or valence $q$.

</i>

<i>

**definition [d]** (*Contravariant Rank = Contravariant Degree*) The number of upper indices on the components of a tensor:

- $T^{\mu_1\ldots\mu_q}_{\ \nu_1\ldots\nu_r}$ has contravariant rank $q$ .

where

- upper indices transform with the Jacobian $\partial x^{\mu}/\partial (x')^{\alpha}$.
- the total rank (order) is $q + r$.

</i>
