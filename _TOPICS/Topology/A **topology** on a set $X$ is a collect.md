A **topology** on a set $X$ is a collection $\mathcal{T}$ of subsets of $X$ that satisfies the following three formal conditions:

1.  **Existence of the Empty Set and the Set Itself**: Both the set $X$ and the empty set $\emptyset$ must be elements of the collection $\mathcal{T}$.
2.  **Closed Under Finite Intersections**: If $U_1, \dots, U_n$ are elements of $\mathcal{T}$, then their intersection $U_1 \cap \dots \cap U_n$ must also be an element of $\mathcal{T}$.
3.  **Closed Under Arbitrary Unions**: If $\{U_\alpha\}_{\alpha \in A}$ is any family of subsets in $\mathcal{T}$ (whether finite or infinite), then their union $\bigcup_{\alpha \in A} U_\alpha$ must also be an element of $\mathcal{T}$.

### **Key Terminology**
*   **Topological Space**: A pair $(X, \mathcal{T})$ consisting of a set $X$ together with a topology $\mathcal{T}$ on $X$ is called a **topological space**.
*   **Open Sets**: Once a set $X$ is endowed with a topology, the subsets belonging to the collection $\mathcal{T}$ are formally called **open subsets** of $X$, or simply **open sets**.
*   **Points**: The elements of the underlying set $X$ are called its **points**.
*   **Closed Sets**: A subset $F \subseteq X$ is considered **closed** if its complement, $X \setminus F$, is an open set.

### **Comparison of Topologies**
Different topologies can be defined on the same set. These are compared using the following terms:
*   **Coarser Topology**: Given two topologies $\mathcal{T}_1$ and $\mathcal{T}_2$ on a set $X$, $\mathcal{T}_1$ is **coarser** than $\mathcal{T}_2$ if $\mathcal{T}_1 \subseteq \mathcal{T}_2$. A coarser topology contains fewer open sets.
*   **Finer Topology**: Conversely, $\mathcal{T}_2$ is **finer** than $\mathcal{T}_1$ if $\mathcal{T}_1 \subseteq \mathcal{T}_2$. A finer topology contains more open sets.

### **Examples of Standard Topologies**
*   **Discrete Topology**: The collection of *all* subsets of $X$. In this topology, every subset is open.
*   **Trivial (or Indiscrete) Topology**: The collection $\{\emptyset, X\}$ consisting only of the empty set and the entire set.
*   **Metric Topology**: A topology generated by a metric (distance function), where

* a set is open if it contains an open ball around each of its points.
*   **Subspace (or Relative) Topology**: If $A \subseteq X$, the subspace topology on $A$ consists of all sets formed by the intersection of $A$ with open sets of $X$.
*   **Finite-Complement (or Cofinite) Topology**: A topology where

* a subset is open if its complement is finite, or if the set is empty.

A **Hausdorff space** (also referred to as a **$T_2$-space**) is a topological space $X$ that satisfies the following formal condition:

Given any pair of **distinct points** $p_1, p_2 \in X$, there exist **disjoint neighborhoods** $U_1$ of $p_1$ and $U_2$ of $p_2$ such that $U_1 \cap U_2 = \emptyset$. This property is summarized by the intuitive idea that "points can be separated by open subsets".







A **topological space** is a pair $(X, \mathcal{T})$ consisting of a set $X$ together with a **topology** $\mathcal{T}$ on $X$. A topology $\mathcal{T}$ is defined as a collection of subsets of $X$ that satisfies the following three formal conditions:

*   **Existence of the empty set and the set itself:** Both the entire set $X$ and the empty set $\emptyset$ must be elements of the collection $\mathcal{T}$.
*   **Closed under finite intersections:** If a finite number of sets $U_1, \dots, U_n$ are elements of $\mathcal{T}$, then their intersection $U_1 \cap \dots \cap U_n$ must also be an element of $\mathcal{T}$.
*   **Closed under arbitrary unions:** If $\{U_\alpha\}_{\alpha \in A}$ is any family of subsets belonging to $\mathcal{T}$ (whether the family is finite or infinite), then their union $\bigcup_{\alpha \in A} U_\alpha$ must also be an element of $\mathcal{T}$.

### Key Related Terminology
*   **Open Sets:** Once a set $X$ is endowed with a topology, the subsets that make up the collection $\mathcal{T}$ are formally called the **open subsets** of $X$, or simply **open sets**.
*   **Points:** The elements of the underlying set $X$ are referred to as its **points**.
*   **Closed Sets:** A subset $F \subseteq X$ is considered **closed** if its complement, $X \setminus F$, is an open set.
*   **Neighborhoods:** A **neighborhood** of a point $p \in X$ is generally defined as any open set containing $p$. However, some texts define a neighborhood more broadly as any subset of $X$ that *contains* an open set containing $p$.

The primary motivation for this abstraction is to provide a qualitative way to study concepts like **convergence** and **continuity** without requiring a quantitative measure of distance, such as a metric. While every metric space automatically inherits a topology (the **metric topology**), not all topological spaces can be defined by a metric.
