# Curve

<i>

**definition [d]** (*Curve = Parametric Curve*) The set of all points $(x,y,z)$ given by continuous parametric equations on an interval $I$:

- $x = f(t)$, \quad $y = g(t)$, \quad $z = h(t)$ , \quad $t \in I$ .

where

- $f, g, h$ are continuous real-valued functions on $I$.
- $t$ is the parameter.
- equivalently, $\mathbf{r}(t) = \langle f(t),\, g(t),\, h(t) \rangle$ traces the curve.

</i>

<i>

**definition [d]** (*Curve = Path*) The image of a continuous map $\boldsymbol{\gamma}: [a,b] \rightarrow \mathbb{R}^{n}$ (or of a continuous vector function $\mathbf{r}(t)$):

- $C = \{\, \boldsymbol{\gamma}(t) \mid t \in [a,b] \,\}$ .

where

- $\boldsymbol{\gamma}$ (or $\mathbf{r}$) is continuous on $[a,b]$.
- $C$ is the set of points traced; the map itself is often called a path.

</i>
