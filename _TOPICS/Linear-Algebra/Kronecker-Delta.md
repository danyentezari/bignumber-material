# Kronecker Delta

<i>

**definition [d]** (*Kronecker Delta = Kronecker Symbol*) The discrete symbol $\delta_{ij}$ defined by

- $\delta_{ij} = \begin{cases} 1 & \text{if } i = j \\ 0 & \text{if } i \neq j \end{cases}$ .

where

- $i, j$ are discrete indices.
- $\delta_{ij}$ are the components of the identity matrix $I$.
- under contraction, $\delta_{ij}$ acts as an index-replacement operator (e.g. $b_{j}\delta_{ij} = b_{i}$).

</i>

<i>

**definition [d]** (*Kronecker Delta = Kronecker Symbol*) The symbol $\delta_{ij}$ (or mixed $\delta^{i}_{\ j}$) defined by

- $\delta_{ij} = 1$ if $i = j$, and $\delta_{ij} = 0$ if $i \neq j$ .

where

- $i, j$ are discrete indices.
- as a type $(1,1)$ tensor, $\delta^{\mu}_{\ \nu}$ is the identity map on vectors / one-forms.
- the components of the identity matrix satisfy $I_{ik} = \delta_{ik}$.

</i>
